# RUSTY-SHTOP

Rusty-SHTOP will be a replacement for HTOP written using rust.  It will display cpu and memory usuage, among other details.

### To Run:
```sh
$ git clone https://github.com/ericmcbride/rusty-shtop.git
$ cargo run
```
